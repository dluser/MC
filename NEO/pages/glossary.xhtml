Glossary

build

build

To compile and package the content in a Habitat project to create an Inkling title. The Inkling title can then be downloaded to and viewed in the Inkling app on supported devices.

caption

caption

Text that describes an image. It appears outside the image, usually below it.

citation

citation

The information that documents the source from which an author took content. Citations that are in a print book’s footnotes and endnotes are stored in Remarks.

enhancement

enhancement

Interactive elements within an Inkling title. The majority enhancements are created using widgets, but in its broadest use, the term can also refer to poptips, inline audio, and other improvements made possible by the Inkling platform.

inline image

inline image

An image that appears on a line of text. Special symbols or equations in the text that cannot be rendered in HTML are rendered as inline images.

pattern

pattern

Inkling term for a snippet of HTML code that includes a reference to CSS. Readymade patterns are available in Habitat's Pattern Picker and Inkling’s Pattern Library.

revision

revision

Every time a project team member saves a change to content in Habitat, that revision of the file is archived with a unique number. A revision automatically generates analyses of the content, and the results are presented in the Reports area of Habitat. Because every revision is given a unique number and is archived, you can view the state of a project at any moment in its history.

sandbox chapter

sandbox chapter

A chapter that is part of a Habitat project but is not meant to be part of your final Inkling title. It can contain examples of content, reusable HTML patterns, or exploratory material. When you create a such chapter in Table of Contents, identify it as a sandbox chapter in the Inspector.

ticket

ticket

Tickets are comments or instructions about content. Create tickets in Habitat Proof mode. Use them to identify errors in content, to suggest improvements, and so on. Assign a ticket to yourself or another team member who can respond to your comment or fix an error.

widget

widget

Widgets enable you to put javascript within <object> tags to provide interactive content in Inkling pages.

XHTML

XHTML

XHTML is a stricter version of HTML. For example, in XHTML all tags must be closed, so use <br /> rather than <br> and <img /> rather than <img>. Habitat's well-formed HTML is a subset of the XHTML standards; Habitat requires code that validates to XHTML standards to ensure that content displays correctly on all supported devices. See Inkling standards for HTML and alsoXHTML Reference.
